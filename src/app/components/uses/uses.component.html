Things I use

I love my MacBook, and with the M1 chips I keep toying with the idea of getting a new MacBook. But since I switched to Ubuntu with i3 a few years ago, I can't imagine doing that anymore. As soon as there is a comparable window manager for MacOS like i3, I'll switch again ;)

My Computer

My primary setup is a Lenovo Carbon X1 (6th gen) with a Samsung Ultra WQHD (S34J550WQU).

I use standard Ubuntu with the window manager i3. The biggest advantage for me: no UX. I have no menu to launch applications, or to switch between applications. It's all designed for me to use shortcuts as efficiently as possible to arrange windows, launch and close applications, and everything else you need. And the transition went extremely smoothly. If you haven't tried it yet, you should!

In my dotfiles you can find all configurations I use.

Productivity

I still use email, specifically Gmail. I've tried different email clients over and over but always fall back to the browser app. I use multiple inboxes to categorize emails, but basically I don't want any emails in my inbox at all. I'm a big fan of Inbox Zero. To minimize GMail to the bare minimum, I use the Chrome extension Simplified GMail.

For notes and a knowledgebase I use respective private Github repositories, basically with Markdown files. For todos I use a Github project on one side and the standard iOS todo app on the other.
